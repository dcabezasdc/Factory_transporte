Programa de ejemplo: Sistema de transporte y embalaje

Este programa Java proporciona un ejemplo básico de un sistema que gestiona el transporte y el embalaje de paquetes.

Funcionalidades principales

Clase abstracta Transporte: Define métodos para calcular el coste total y determinar el tipo de embalaje.
Clase Camion: Implementa los métodos de la clase abstracta para el transporte en camión.
Clase Bicicleta: Implementa los métodos de la clase abstracta para el transporte en bicicleta.
Clase Main: Contiene el método principal para probar la aplicación.
Ejecución del programa

Compila el archivo Java utilizando el comando javac Main.java.
Ejecuta el archivo compilado con el comando java Main.
Observa la salida en la consola para ver el coste total y el tipo de embalaje para un envío en un camión y en una bicicleta.
